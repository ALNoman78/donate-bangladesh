Important FAQ

What is the DOM model ?

The DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents the document structure as a tree of objects, where each node corresponds to elements, attributes, or text. Developers use the DOM to access, modify, and interact with elements in a webpage using languages like JavaScript. The DOM allows dynamic updates to the page's content, structure, and styles. Events like clicks or keypresses can be handled using DOM event listeners. The DOM provides a way to build interactive and responsive web applications

How do you select element form DOM ?

To select elements from the DOM, JavaScript provides several methods. Here are some commonly used ones:

  1. getElementById(id)
  2. getElementsByClassName(className)
  3. getElementsByTagName(tagName)
  4. querySelector(selector)
  5. querySelectorAll(selector)
  6. getElementsByName(name)
What is event delegation in the context of DOM , and why it's useful ?

Event delegation is a technique in JavaScript where you add a single event listener to a parent element instead of adding individual event listeners to each child element. The event listener on the parent captures events that bubble up from its children due to event propagation, specifically event bubbling. This allows handling events for dynamically added elements as well.

How do you manipulate an Element's attributes and style using DOM ?
To manipulate an element's attributes and styles using the DOM, you can use JavaScript methods and properties. Here's how:

Attribute

1. getAttribute() – Retrieves the value of a specified attribute, 2. setAttribute() – Sets the value of an attribute, 3. removeAttribute() – Removes an attribute from an element

Style

1. Using style property – Directly change CSS properties of an element, 2.Using classList – Add, remove, or toggle CSS, 3. Setting multiple styles at once – You can also use cssText to assign multiple styles
